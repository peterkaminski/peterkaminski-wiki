ChatGPT

ChatGPT is a conversational AI developed by OpenAI, first made available to the general public in November 2022.

Resources

  • ChatGPT Exporter - a userscript for Tampermonkey that exports one or many conversations from the web front end.
  • chap - a terminal UI front end for the ChatGPT API that uses files to save and load conversations.
  • Sylph - a simple proof-of-concept file-based front end for the ChatGPT API originally created with GPT-4.
